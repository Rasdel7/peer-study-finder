# Peer Study Group Finder 🤝

A social matching app that connects students with
compatible study partners based on subjects,
schedule and learning style.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Weighted match-scoring algorithm (subjects,
  time, learning style, year proximity)
- Create/edit your own study profile
- Browse all registered students with filters
- Subject demand analytics
- Visual match breakdown per candidate

## How Matching Works
Score out of 100: Subject overlap (50pts) +
Schedule match (25pts) + Learning style (15pts) +
Year proximity (10pts)

## Tools Used
- Python, Streamlit, Plotly, Pandas

## How to Run Locally
pip install streamlit pandas numpy plotly
streamlit run app.py
